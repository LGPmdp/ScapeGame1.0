Escape del hospital

Pulse para comenzar

Este es un juego que tiene como objetivo el intentar encontrar la salida. Para cada opción que tomes habrá una consecuencia. Intenta elegir bien... una mala decisión puede llevarte a la muerte.
Suerte para encontrar la salida del hospital!

El juego fue hecho en base a imaginación propia, utilizando los conceptos básicos del lenguaje de programación HTML y CSS, con el objetivo de entretener
